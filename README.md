# Computer Limiter
- Allows the user to create warnings and dialogue boxes that forces them to take a break when using the computer. 

## Instructions to use: 
1. Clone File:
2. If In Linux:
    * run *python (or python3 ) test.py &*
    If in Windows:
    - run *start /b python main.py*
